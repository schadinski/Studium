Web Tec Nacht 2016


Katharina Schwab

In diesem Projekt liegt der Schwerpunkt auf dem Umgang mit Canvas als multimedialer Leinwand. Es soll eine Animation erstellt werden, auf die der User mindestens 2 Möglichkeiten der Einflussnahme hat. Des weiteren sollen dynamische und statische Objekte verwendet werden.
Dies wird hier mittels einer Feuerwehranimation umgesetzt, die es ermöglicht selbst zu bestimmen, welche Brände gelöscht werden sollen. Auch die Geschwindigkeit des umherfahrenden Feuerwehrautos kann mittels Eingabe bestimmt werden. Der User hat die Möglichkeit zwischen einer Animation bei Tag und einer bei Nacht zu wählen. Wird auf ein Haus geklickt beginnt der Löschvorgang. Wenn offt genug geklickt wird, erlischt das Feuer in diesem Haus.
